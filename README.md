# Family Fluke

A Family Feud knock-off for Luke's birthday. Pure static site — open `index.html` in a browser and play.

## Controls

| Key | Action |
|-----|--------|
| 1–6 | Reveal answer |
| X | Strike |
| Left/Right | Change question |
| T | Toggle team |
| Space | Award points to active team |

Edit `questions.js` to customise the questions.
